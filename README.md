# 経済・デジタル指標分析ダッシュボード (2005-2025)

主要国の名目GDP、一人当たりGDP、ICT比率、デジタル収支などの推移を可視化するインタラクティブ・ダッシュボードです。

## 1. 開発の目的と可視化の意図

本プロジェクトは、過去20年間のグローバル経済の変遷とデジタル化の進展を「大局的に把握すること」を目的としています。

* **USD（米ドル建て）での表示**:
* 目的：**国際比較**。
* 同一基準（ドル）で各国の経済規模や生産性を並べることで、世界経済における相対的な立ち位置や成長スピードの差を可視化します。


* **JPY（円建て）での表示**:
* 目的：**国内における絶対評価・実力値の確認**。
* 為替変動の影響を除いた、日本国内での経済規模の推移を直感的に理解するために用意しています。



## 2. データの構成と算出根拠

本ダッシュボードでは、以下の主要な指標を表示します。

| 指標名 | 単位 | 内容・算出 |
| --- | --- | --- |
| **名目GDP** | 兆USD / 兆円 | 各国の経済活動の総量。インフレ率を含む市場価格。 |
| **一人当たりGDP** | USD | 経済的な豊かさや労働生産性の目安。 |
| **ICT産業の対GDP比率** | % | GDPに占める情報通信産業の割合（デジタル化の深度）。 |
| **デジタル収支** | Billion USD / 兆円 | サービス収支のうち、デジタル関連取引（著作権、ITサービス等）の収支。 |

### 主なデータソース

実績値については各国の統計局データを参照し、直近および将来予測についてはIMFのデータベースを活用しています。

* **米国**: BEA (Bureau of Economic Analysis)
* **ドイツ**: Destatis (Statistisches Bundesamt)
* **中国**: NBS (National Bureau of Statistics of China)
* **インド**: MoSPI (Ministry of Statistics and Programme Implementation)
* **日本**: 内閣府（国民経済計算）
* **予測値 (2024-2025)**: IMF World Economic Outlook (October 2024)

## 4. Disclaimer（免責事項・注意書き）

### データ整理における生成AIの活用

本ダッシュボードのデータセットは、膨大な公的統計および報告書から**生成AIを用いて収集・整理・補完**を行っています。AIによる効率的なデータ統合を行っていますが、以下の点にご注意ください。

* **数値の精緻性**: 通貨換算（為替レートの適用時期）や推計手法の違いにより、一次統計と細部で数値が異なる場合があります。
* **名目GDPについて**: すべて「名目値」を使用しています。物価変動の影響を除いた実質的な成長率とは異なる動きをすることがあります。
* **用途**: 本ツールは学術的・実務的な厳密な引用を目的としたものではなく、マクロ経済の大きなトレンドを視覚的に理解するためのプロトタイプとしてご利用ください。
